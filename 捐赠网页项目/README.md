# NDP 新捐赠平台

## 项目简介
它的由来是作者之前刷到过一个受赠者的视频，下方很多人都在评论说现在不敢捐钱了，因为害怕钱到不了真正需要的人手中，所以俺就想着设计一款受赠者与捐赠者可以直接交流的网站，这样是不是就可以解决大家的担忧了呢~
一个为公益组织设计的捐赠平台，直接连接捐赠者与受赠者，确保每一份善意都能精准送达。

## 功能列表

### 核心功能
- ✅ **用户注册与登录**：支持捐赠者和受赠者两种角色注册
- ✅ **头像上传与修改**：用户可自定义个人头像，选个美美哒头像心情都变好~
- ✅ **受赠者注册**：支持受赠者上传图片并详细说明自身情况，管理员会进行严格的审核
- ✅ **捐赠功能**：支持金钱和物品捐赠（当然，现在网站的金钱捐赠功能还未彻底实现）
- ✅ **聊天功能**：捐赠者与受赠者可直接交流，捐赠者跟管理员也可以直接交流，受赠者想联系管理员可以通过下面的“联系我们”
- ✅ **个人主页**：查看个人信息和捐赠记录，还可以查看聊天消息~
- ✅ **管理员后台**：审核受赠者、管理用户

### 特色功能
- 实时聊天系统
- 捐赠记录查询
- 受赠者信息展示
- 响应式设计，支持移动端

## 技术栈

### 前端
- HTML5 + CSS3
- JavaScript (原生)
- LocalStorage (本地存储)

### 后端
- Node.js + Express
- MongoDB (数据库)
- JWT (身份验证)
- Socket.io (实时通信)

### 依赖库
- bcrypt (密码加密)
- cors (跨域支持)
- dotenv (环境变量管理)
- mongoose (MongoDB ORM)
- multer (文件上传)
